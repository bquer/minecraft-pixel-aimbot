# minecraft-pixel-aimbot
Undedectable minecraft pixel aimbot with gui
# Tested On
- Windows 10
- Windows 11
- Ubuntu 22.04.3 LTS
# Features
- AimBot & Hack GUI ( Used Tkinter )
- Key bind ( 4 aimbot on/off)
# Note
You need to open hitboxes with F3+B / if u using laptop Fn+F3+B
